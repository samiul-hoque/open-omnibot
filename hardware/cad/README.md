## Source Files

Editable source files are in `source/` directory:
- `.step` - Universal CAD format
- `.f3d` - Fusion 360 native format

## Print Settings

- Layer height: 0.2mm
- Infill: 20-30%
- Supports: As indicated per part
- Material: PLA (indoor) or PETG (more durable)
